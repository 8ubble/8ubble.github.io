一些易错题：

No.1 从这四个选项选出不同的一个：

A、JQuery B、Node.js C、Prototype D、CommonJS

当时真没看懂题目什么意思：第一个很常见，是个事件库。Node.js是一套工具包。没怎么听过CommonJS，但是我知道Prototype 属性是向对象添加属性和方法，然后我就选了这个。但是，答案是D。因为：

Prototype.js是由Sam Stephenson写的一个JavaScript库。。。

CommonJS是一套规范，NodeJS是这种规范的实现。

好吧……

No.2 以下哪一条JavaScript语句会产生运行错误：

A、var obj=(); B、var obj=[]; C、var obj={}; D、var obj=/ /;

在浏览器里调试了一下答案A，会显示Uncaught SyntaxError: Unexpected token )，然后答案选A。而D居然是声明正则对象。。。
（原来D答案这中间是有空格的）

No.7 Ajax和Flash相比，有什么优劣？

  • Ajax的优点：开放性、易用性和易于开发，但是有可能破坏浏览器的后退功能

  • Flash多用于多媒体处理，可以更容易的调用浏览器以外的外部资源。但是文件体积有可能挺大……

No.8 下面有关浏览器中使用js跨域获取数据的描述，说法错误的是？

A、域名、端口相同，协议不同，属于相同的域

B、js可以用jsonp进行跨域

C、通过修改document.domain来跨子域

D、使用window.name来进行跨域

答案是A，域名、端口相同，协议有一改动的地方就会被认为是不同的域

No.11 关于IE、FF下面脚本的区别描述错误的是？

A、innerText IE支持，firefox不支持

B、document.createElement firefox支持，IE不支持

C、setAttribute(‘class’ , ‘styleClass’) firefox支持，IE支持

D、用setAttribute设置事件，firefox不支持，IE支持

答案是BD，具体参考SD9017: Firefox 不支持 DOM 对象的 outerHTML、innerText、outerText 属性、SD9010: 仅 IE 中的 createElement 方法支持传入 HTML String 做参数、SD9006: IE 混淆了 DOM 对象属性（property）及 HTML 标签属性（attribute），造成了对 setAttribute、getAttribute 的不正确实现等条目

反正都是官方标准

No.15 语句var arr=[a,b,c,d];执行后，数组arr中每项都是一个整数，下面得到其中最大整数语句正确的是哪几项？

A、Math.max(arr)

B、Math.max(arr[0], arr[1], arr[2], arr[3])

C、Math.max.call(Math, arr[0], arr[1], arr[2], arr[3])

D、Math.max.apply(Math,arr)

答案是BCD，A中的函数 Math.max(x);的参数是Number类型，可以是小数，整数，正数，负数或者是0.如果不是上面所述类型就会返回NaN.
